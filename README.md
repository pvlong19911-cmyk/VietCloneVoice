## 📦 Hướng dẫn cài đặt
1. Tải file `.rar` về máy
2. Giải nén bằng WinRAR hoặc 7-Zip
3. Mở thư mục `VietCloneVoice`
4. Chạy `VietCloneVoice.exe`
